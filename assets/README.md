# Assets

Ce dossier contient les assets de l'application AtlasGo.

## Images requises

Placez les fichiers suivants dans ce dossier :

- `icon.png` - Icône de l'application (1024x1024px)
- `splash.png` - Écran de démarrage (1284x2778px)
- `adaptive-icon.png` - Icône adaptative Android (1024x1024px)
- `favicon.png` - Favicon web (32x32px)

## Couleurs recommandées

- **Couleur principale** : #0A66C2 (Bleu Égée)
- **Couleur d'accent** : #D4A017 (Or)
- **Arrière-plan** : #F7F9FC (Light) / #0B1220 (Dark)

## Formats supportés

- PNG (recommandé)
- JPG (pour les photos)
- SVG (pour les icônes vectorielles)

## Tailles recommandées

- **Icon** : 1024x1024px
- **Splash** : 1284x2778px (iPhone 12 Pro Max)
- **Adaptive Icon** : 1024x1024px
- **Favicon** : 32x32px, 64x64px, 128x128px
